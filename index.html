Cheat Sheet

IT · Computer Science · Data Science · DevOps · Networking

⌕
Programming & Scripting
.cpp
C++ Source
Compiled language
Compile
$ g++ -o output file.cpp
Run
$ ./output
Compile + Run (one-liner)
$ g++ file.cpp -o out && ./out
Install: sudo apt install g++
.c
C Source
Compiled language
Compile
$ gcc -o output file.c
Run
$ ./output
Install: sudo apt install gcc
.py
Python Script
Interpreted language
Run
$ python3 file.py
Interactive mode
$ python3 -i file.py
With venv
$ source venv/bin/activate && python3 file.py
Install: sudo apt install python3
.js
JavaScript
Node.js runtime
Run with Node
$ node file.js
Watch mode (nodemon)
$ nodemon file.js
Install: sudo apt install nodejs
.ts
TypeScript
Compiled to JS
Compile + Run
$ tsc file.ts && node file.js
Run directly (ts-node)
$ ts-node file.ts
Run with Deno
$ deno run file.ts
.sh
Shell Script
Bash / POSIX shell
Run directly
$ bash file.sh
Make executable + run
$ chmod +x file.sh && ./file.sh
Debug mode
$ bash -x file.sh
.java
Java Source
Compiled to bytecode
Compile
$ javac file.java
Run (no .class extension)
$ java ClassName
Install: sudo apt install default-jdk
.go
Go Source
Compiled language
Run directly
$ go run file.go
Build binary
$ go build -o output file.go
Install: sudo apt install golang
.rs
Rust Source
Systems language
Compile
$ rustc file.rs
Run (via Cargo)
$ cargo run
Install: curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
.rb
Ruby Script
Interpreted language
Run
$ ruby file.rb
IRB interactive
$ irb -r ./file.rb
Install: sudo apt install ruby
.pl
Perl Script
Interpreted language
Run
$ perl file.pl
Install: sudo apt install perl
.php
PHP Script
Server-side language
Run CLI
$ php file.php
Built-in web server
$ php -S localhost:8000
Install: sudo apt install php
Web & Markup
.html
HTML File
Web markup
Open in browser
$ xdg-open file.html
Serve locally (Python)
$ python3 -m http.server 8080
Serve (Node live-server)
$ npx live-server .
.css
CSS Stylesheet
Style sheet language
Lint with stylelint
$ npx stylelint file.css
Minify
$ npx csso file.css -o file.min.css
.xml
XML File
Markup / config data
Pretty-print
$ xmllint --format file.xml
Validate
$ xmllint --noout file.xml
Query with XPath
$ xmllint --xpath "//tag" file.xml
Install: sudo apt install libxml2-utils
.json
JSON File
Data interchange format
Pretty-print
$ cat file.json | python3 -m json.tool
Query with jq
$ jq '.' file.json
Install jq: sudo apt install jq
.yaml / .yml
YAML Config
Data serialization
Validate / parse
$ python3 -c "import yaml,sys; yaml.safe_load(open('file.yaml'))"
Lint with yamllint
$ yamllint file.yaml
Install: pip install yamllint
.md
Markdown File
Docs / README
View rendered (glow)
$ glow file.md
Convert to HTML
$ pandoc file.md -o file.html
Install glow: snap install glow
Documents & Text
.txt
Plain Text
General purpose text
View
$ cat file.txt
Scroll through (pager)
$ less file.txt
Edit
$ nano file.txt
Word count
$ wc -w file.txt
.pdf
PDF Document
Portable Document Format
Open in viewer
$ evince file.pdf
Extract text
$ pdftotext file.pdf output.txt
Merge PDFs
$ pdfunite a.pdf b.pdf out.pdf
Info / metadata
$ pdfinfo file.pdf
Install: sudo apt install poppler-utils evince
.docx
Word Document
Microsoft Word format
Open in LibreOffice
$ libreoffice file.docx
Convert to PDF
$ libreoffice --headless --convert-to pdf file.docx
Extract text (pandoc)
$ pandoc file.docx -t plain
Install: sudo apt install libreoffice
.xlsx / .xls
Excel Spreadsheet
Tabular data
Open in LibreOffice Calc
$ libreoffice --calc file.xlsx
Convert to CSV
$ libreoffice --headless --convert-to csv file.xlsx
Python: pip install openpyxl
.csv
CSV File
Comma-separated values
View formatted
$ column -t -s, file.csv | less
View with csvkit
$ csvlook file.csv
Count rows
$ wc -l file.csv
Install: pip install csvkit
Data Science & Machine Learning
.ipynb
Jupyter Notebook
Interactive data science
Launch Jupyter Lab
$ jupyter lab file.ipynb
Run all cells headless
$ jupyter nbconvert --to notebook --execute file.ipynb
Convert to Python
$ jupyter nbconvert --to script file.ipynb
Install: pip install jupyterlab
.R / .r
R Script
Statistical computing
Run script
$ Rscript file.R
Interactive R console
$ R
Install: sudo apt install r-base
.parquet
Apache Parquet
Columnar data format
Inspect schema
$ python3 -c "import pandas as pd; print(pd.read_parquet('file.parquet').info())"
View with DuckDB
$ duckdb -c "SELECT * FROM 'file.parquet' LIMIT 10"
Install: pip install pandas pyarrow duckdb
.h5 / .hdf5
HDF5 File
Scientific / ML data
List structure
$ h5ls -r file.h5
Dump to text
$ h5dump file.h5 | less
Install: sudo apt install hdf5-tools
.pkl
Python Pickle
Serialized Python object
Inspect
$ python3 -c "import pickle; obj=pickle.load(open('file.pkl','rb')); print(type(obj))"
⚠ Never unpickle files from untrusted sources
.npy / .npz
NumPy Array
Numerical data arrays
Load and inspect
$ python3 -c "import numpy as np; a=np.load('file.npy'); print(a.shape, a.dtype)"
Databases & Configuration
.sql
SQL Script
Database query / DDL
Run on SQLite
$ sqlite3 mydb.db < file.sql
Run on PostgreSQL
$ psql -U user -d dbname -f file.sql
Run on MySQL
$ mysql -u user -p dbname < file.sql
.db / .sqlite
SQLite Database
Embedded database file
Open interactive shell
$ sqlite3 file.db
List tables
$ sqlite3 file.db ".tables"
Query to CSV
$ sqlite3 -csv file.db "SELECT * FROM t"
.env
Environment File
Config variables
Export all vars to shell
$ export $(cat .env | xargs)
Source into current shell
$ source .env
.toml / .ini
Config Files
App configuration
View
$ cat file.toml
Validate TOML
$ python3 -c "import tomllib; tomllib.load(open('file.toml','rb'))"
Dockerfile
Dockerfile
Container definition
Build image
$ docker build -t myimage .
Build + run
$ docker build -t img . && docker run -it img
Lint (hadolint)
$ hadolint Dockerfile
Makefile
Makefile
Build automation
Run default target
$ make
Run specific target
$ make target_name
List all targets
$ make -qp | awk '/^[^#\t].*:/{print $1}'
System, Networking & DevOps
(binary)
ELF Binary / Executable
Compiled program
Make executable + run
$ chmod +x ./program && ./program
File info
$ file ./program
List dynamic dependencies
$ ldd ./program
.tar.gz / .zip
Archives
Compressed file bundles
Extract .tar.gz
$ tar -xzvf file.tar.gz
Extract .zip
$ unzip file.zip
Create .tar.gz
$ tar -czvf out.tar.gz ./folder
List contents (no extract)
$ tar -tzvf file.tar.gz
.log
Log File
System / app logs
Follow live output
$ tail -f file.log
Search with grep
$ grep -i "error" file.log
Last 100 lines
$ tail -n 100 file.log
.pem / .crt
SSL Certificate / Key
TLS / PKI
View certificate info
$ openssl x509 -in file.crt -text -noout
Check expiry
$ openssl x509 -in file.pem -enddate -noout
Install: sudo apt install openssl
.pcap
Packet Capture
Network traffic
Read in terminal
$ tcpdump -r file.pcap
Open in Wireshark
$ wireshark file.pcap
Filter with tshark
$ tshark -r file.pcap -Y "http"
Install: sudo apt install wireshark tshark
playbook.yml
Ansible Playbook
Infrastructure automation
Run playbook
$ ansible-playbook playbook.yml
Dry-run (check mode)
$ ansible-playbook playbook.yml --check
Install: pip install ansible
.tf
Terraform Config
Infrastructure as code
Initialize
$ terraform init
Preview changes
$ terraform plan
Apply changes
$ terraform apply
.awk
AWK Script
Text stream processing
Run on file
$ awk -f script.awk data.txt
Quick one-liner
$ awk '{print $1}' data.txt
